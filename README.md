# Roommate Finder
A web application that allow users to find available rooms or post room listings with search and cost filtering.

## Features
- User registration and login
- Post room listings
- Search rooms
- Filter by cost
  
## Tech Stack
- React
- Node.js
- REST API
  
## Installation
git clone https://github.com/wesalmagdi/Roommate-Finder.git
cd Roommate-Finder
npm install
npm start

## Usage
Users can browse rooms, apply filters, and post listings.
