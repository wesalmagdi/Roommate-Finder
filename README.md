# Roommate Finder
Full-stack web application designed to allow users to find available rooms for rent or post rooms easily. The platform allows users to search and filter room listings based on budget, gender preference, wifi availability, pets permission, AC and location, making the process of finding suitable shared housing faster and more efficient. 

## Features
- User registration and authentication
- Create and manage room listings
- Search rooms
- Filter by budget, location, Wifi, pets permissin, AC availability or gender
- Responsive interface
  
## Tech Stack
- Frontend: React
- Backend: Node.js
- API: REST API
- Authentication: JSON Web Tokens (JWT)
  
## Installation
git clone https://github.com/wesalmagdi/Roommate-Finder.git
cd Roommate-Finder
npm install
npm start

## Usage
- Register a new account or log in
- Browse available room listings
- Use filters to narrow results
- Create a room listing if you have a room available

## Security
- Passwords hashed before storage.
- JWT verification for secure authentication.
- Environment variables used for sensitive data.
- Prevent multiple logins using valid tokens.
